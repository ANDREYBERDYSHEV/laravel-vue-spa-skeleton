## Install
- `git clone https://github.com/Yurich84/laravel-vue-spa-skeleton.git`
- `cd laravel-vue-spa-sceleton`
- `composer install`
- set your DB credentials in `.env`
- `php artisan migrate`
- `yarn`
- `yarn dev`
- `php artisan serve` and open browser
